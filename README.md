## dependencies
* Apollo Server,
* GraphQL,
* bcrypt,
* JSONWebToken,
* mongoose,
* nodemon (dev).

## usage
make "config.js" in project's main folder and put your credentials here
```javascript
module.exports = {
  MONGODB: "<your mongodb>",
  KEY: "<your JSONWebToken secret key>",
};
```

after getting all of the dependencies (also nodemon) run
```
npm start
```
in your terminal
